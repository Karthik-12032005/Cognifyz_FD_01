# Student Registration Form

A responsive and interactive Student Registration Form built using HTML, CSS, Node.js, Express.js, and EJS. This project demonstrates server-side form handling, dynamic rendering, and modern UI design in a full stack web application.

---

## 🚀 Features

- User-friendly registration form
- Beautiful modern UI
- Server-side form handling
- Dynamic rendering using EJS
- Responsive design
- Built with Express.js

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Node.js
- Express.js
- EJS

---
